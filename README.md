# waveform-truth-engine

Deterministic waveform auditing and instant remastering (non-generative).

## Features
- High-density waveform frame analysis
- Shazam-style invariant landmark checks
- Confidence scoring per frame
- Reversible enhancement (no re-synthesis)
- Offline CLI

## Install
pip install -r requirements.txt

## Run
python cli.py input.mp3 --out output.wav --enhance
